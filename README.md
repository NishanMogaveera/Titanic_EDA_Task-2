# 🚢 Titanic Exploratory Data Analysis (EDA)


## 📌 Project Objective
The objective of this project is to analyze the Titanic passenger dataset and identify the major factors that influenced passenger survival.

The analysis focuses on:
- Missing value handling
- Summary statistics
- Distribution analysis
- Correlation analysis
- Outlier detection
- Multivariate visualization
- Insight generation


## 📂 Dataset Used
- **Dataset:** Titanic Dataset
- **Rows:** 891
- **Columns:** 12
- **Type:** Mixed numerical + categorical data


## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 EDA Steps Performed
### ✅ 1. Data Inspection
- Loaded dataset using pandas
- Checked shape, datatypes, missing values

### ✅ 2. Summary Statistics
- Used `describe()` for statistical summary

### ✅ 3. Distribution Analysis
- Age distribution histogram
- Fare distribution boxplot
- Survival countplots

### ✅ 4. Correlation Analysis
- Correlation heatmap
- Survival relationship with Fare and Passenger Class

### ✅ 5. Outlier Detection
- Fare boxplot used to detect extreme ticket prices
- Identified luxury first-class fare outliers

### ✅ 6. Advanced Visualization
- Pairplot for multivariate analysis

---

## 📊 Key Insights
- Female passengers had significantly higher survival rates
- 1st class passengers had the highest survival
- Higher fares were positively related to survival
- Most passengers were aged between 20–40
- Passenger class strongly impacts fare and survival
